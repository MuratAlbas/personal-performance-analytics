# Personal Performance Analytics & BI Case Study

A Business Intelligence case study focused on transforming 24 days of personal fitness data into actionable performance insights. This project treats a training routine as a production process, analyzing metrics like workload, intensity, and recovery stability.

## 📊 Key Highlights
- **Total Work Growth:** Identified a **28% net increase** in total work volume over an 11-session tracking period.
- **Statistical Correlation:** Modeled the relationship between intensity (Weight) and output (Repetitions), yielding a stable $R^2 = 0.56$ correlation.
- **Variance Detection:** Applied Box and Whisker plots to identify consistency and fluctuations across multiple exercises (Hack Squat, Machine Chest Press, Cable Row).

## 📈 Methodology & Metrics
Performance is measured through three core operational categories:
1. **Weights (kg):** Represents the difficulty level of each task.
2. **Repetitions:** Shows the amount of work done per session.
3. **Total Work (Volume):** The primary success metric to track progressive overload.

### Distribution Analysis (Example Insight)
Using **Box and Whisker Plots**, the analysis revealed different performance patterns:
- **Machine Chest Press:** Most reliable and steady results with minimal session-to-session fluctuation.
- **Hack Squat:** Highest overall work volume but showed wider range of results, indicating sensitivity to daily fatigue levels.

## Future Improvements
If expanded into a professional fitness application or a pro-athlete reporting tool, the next steps would include:
- **Tracking More Factors:** Integrating data on sleep quality, daily calories, and stress levels to explain the remaining 44% of performance variance.
- **Live Dashboarding:** Transitioning from static sheets to a live dashboard (Power BI or Tableau) for active KPI tracking.
